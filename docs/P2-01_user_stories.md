# P2-01 — User Stories

User stories for v1 of the Android app. Written from the perspective of Alok, the sole user. Grouped by feature area.

---

## 1. Quick Log

- As Alok, I want to open the app and log any metric (habit value, task update, job application) in under 30 seconds, so that daily logging doesn't feel like a chore.
- As Alok, I want the quick-log screen to show me what I haven't logged today, so that I don't miss anything without being forced into a checklist.

---

## 2. Project Tracker — Setup

- As Alok, I want to create a new project and choose from pre-built templates (Habit Routine, Task Board, Weekly Review, Job Hunt), so that I'm not starting from a blank canvas.
- As Alok, I want to customise the metrics tracked in any project (add/remove/rename fields, set units), so that the project fits my exact workflow.
- As Alok, I want to configure the points formula for each project individually, so that I can reward high-effort projects more than low-effort ones.
- As Alok, I want to set a threshold/target value for each metric in a project, so that I know when I've hit the goal for the day.

---

## 3. Habit Routine Template

- As Alok, I want to log each habit with its specific value (e.g., wake-up time in HH:MM, meditation in mins, pushups as count), so that the log is precise not just a yes/no.
- As Alok, I want to mark a day as "skipped" for a habit without it counting against me, so that I can use my 2 allowed skip days per week guilt-free.
- As Alok, I want to see my average performance per habit over time, so that I know which habits I'm struggling with.
- As Alok, I want to see what % of days I met the threshold for each habit, so that I have an honest picture of consistency.

---

## 4. Task Board Template (Daily Tracker)

- As Alok, I want to create tasks with a name, description, assignee, start date, and priority, so that I have full context on each task.
- As Alok, I want to set a target number of items and log progress against it, so that I can track incremental completion.
- As Alok, I want tasks to be auto-scored based on priority and completion %, so that I can see which tasks need attention without manually ranking them.
- As Alok, I want to update a task's status (Not Yet Started / In Progress / On Hold / Complete), so that I always have a live view of what's active.

---

## 5. Weekly Review Template

- As Alok, I want the Weekly Review to automatically aggregate data from my daily logs (completion %, hours logged, points earned), so that I don't have to fill it in manually.
- As Alok, I want to see a weekly summary comparing planned hours vs completed hours, so that I can identify if I'm over or under-committing.
- As Alok, I want to see my average earnings/points for the week, so that I can track motivational momentum.

---

## 6. Job Hunt Template

- As Alok, I want to log a job application with the company name, job URL, recruiter name and LinkedIn, and the date applied, so that I have a clean record of my pipeline.
- As Alok, I want to track which job portals I'm using, so that I can identify where I'm getting traction.
- As Alok, I want to update the status of each application (Applied / In Progress / Offer / Rejected), so that I always know where each company stands.

---

## 7. Rewards

- As Alok, I want to define my own rewards (e.g., "Movie Night", "Cheat Meal") and set a point threshold for each, so that I'm motivated by goals I actually care about.
- As Alok, I want to see my current point balance and how close I am to each reward, so that I can feel progress building.
- As Alok, I want to manually claim a reward when I've earned it, so that the act of claiming feels intentional and satisfying.
- As Alok, I want unclaimed rewards to stay available indefinitely, so that I don't feel pressured to use them immediately.
- As Alok, I want to see a history of rewards I've claimed, so that I can look back on my wins.

---

## 8. Streak Mechanic

- As Alok, I want to see my current streak every time I open the app, so that maintaining it feels rewarding and breaking it feels meaningful.
- As Alok, I want my streak to count consecutive days where I logged at least one entry, so that the bar is achievable but still requires daily intent.
- As Alok, I want to see my longest streak ever alongside my current streak, so that I have a personal record to beat.
- As Alok, I want a celebratory popup when I hit a streak milestone (e.g., 7 days, 30 days, 100 days), so that the app acknowledges my consistency.
